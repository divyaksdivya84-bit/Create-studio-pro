# Creator Studio Pro

🚀 Live Demo: https://creator-studio-pro-pz78agat9xtgukpbiyyqjh.streamlit.app/

Creator Studio Pro is a video editing web application built using Python, Streamlit, OpenCV, and FFmpeg.

## Features

- Auto Caption Generation (.SRT subtitles)
- Video Trimming
- Audio Replacement
- Watermarking
- Video Encoding
- Video Processing Automation

## Tech Stack

- Python
- Streamlit
- OpenCV
- FFmpeg

## Achievements

- Reduced manual effort by 60%
- Reduced processing time by 40%
- 1st Place Winner in Hackathon among 80+ teams

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
